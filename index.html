SmartBrain – Face Recognition App

A full-stack web application that allows users to register, log in, and submit image URLs to detect faces using an external AI API.


SmartBrain is a production-ready full-stack web application where users can create accounts, log in, and submit image URLs to detect human faces. The frontend is built in React and communicates with a custom backend API developed with Node.js and Express. User data and image submissions are stored in a PostgreSQL database. The app integrates with a third-party face detection API, allowing real-time image analysis. Because the backend is deployed on Render, the first request may take a few seconds to respond due to server cold start — after that, performance becomes normal.
